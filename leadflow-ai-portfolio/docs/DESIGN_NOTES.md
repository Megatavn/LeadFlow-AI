# Design Notes — Taste Skill Direction

LeadFlow AI v2 follows a calm premium mobile CRM style:

- Warm off-white background instead of generic blue/purple AI dashboard
- Strong hero card with clear product positioning
- Large readable Vietnamese UI
- Rounded cards, soft shadows, generous spacing
- Bottom navigation designed for one-hand phone usage
- Sparse metrics: only what helps the user decide the next action
- Rule explanations in AI Signal Stack so the demo feels credible

The product should communicate: **simple, useful, business-aware, and portfolio-ready**.
